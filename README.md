# 📚 LearnQuest — Gamified Learning Hub

## 📌 Project Overview
LearnQuest is an interactive web-based quiz application designed to make learning fun and engaging through gamification. The platform allows users to test their knowledge across multiple subjects while earning points, XP, badges, and tracking their progress.

---

## 🎯 Objective
The objective of this project is to:
- Create an engaging quiz platform using HTML, CSS, and JavaScript  
- Implement gamification elements like XP, levels, streaks, and rewards  
- Enhance user interaction through dynamic UI and real-time feedback  

---

## 🛠️ Technologies Used
- **HTML5** – Structure of the web application  
- **CSS3** – Styling and responsive design  
- **JavaScript (jQuery + Vanilla JS)** – Logic and interactivity  

---

## ✨ Features
- 📖 Multiple subjects (Math, Science, History, Computer Science, English, Geography)  
- ⏱️ Timer-based quiz system  
- 🎯 Score and XP tracking  
- 🔥 Streak system for consecutive correct answers  
- 🏆 Achievements and badge system  
- 📊 Leaderboard ranking  
- 👤 User profile customization  
- 💡 Hint system (with point deduction)  
- 📱 Responsive design  

---

## 🚀 Unique Feature
### 🔥 Real-Time Combo Multiplier System

This project includes a unique gamification feature called the **Real-Time Combo Multiplier System**:

- Users earn increasing **combo multipliers (x2 to x6)** for consecutive correct answers  
- Each correct answer increases the multiplier and boosts points  
- Bonus XP is awarded based on the combo level  
- The combo resets when a wrong answer is selected  
- A **live combo bar** visually shows the active multiplier  

This feature makes the quiz more engaging and different from traditional quiz applications.
